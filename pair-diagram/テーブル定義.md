TypeScriptとGoで使用するために、各テーブルのカラム名をPascalCaseとsnake_caseの二つの形式で表します。

### ログイン機能

**テーブル名：users**

| PascalCase       | snake_case        | データ型  | 説明                       |
| ---------------- | ----------------- | --------- | -------------------------- |
| Id               | id                | INT       | ユーザーID（主キー）       |
| Username         | username          | VARCHAR   | ユーザー名                 |
| Email            | email             | VARCHAR   | メールアドレス（ユニーク） |
| Password         | password          | VARCHAR   | ハッシュ化されたパスワード |
| Role             | role              | VARCHAR   | 権限（例：admin, user）    |
| CreatedAt        | created_at        | TIMESTAMP | 作成日時                   |
| UpdatedAt        | updated_at        | TIMESTAMP | 更新日時                   |
