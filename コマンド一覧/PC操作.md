以下に、Windows、Mac、Linuxそれぞれの主要なPC操作のショートカットキーの一覧をまとめました。

### システム全般

| 操作                   | Windows                      | Mac                             | Linux                         |
|------------------------|------------------------------|---------------------------------|-------------------------------|
| コピー                 | `Ctrl` + `C`                 | `Cmd` + `C`                     | `Ctrl` + `C`                  |
| 貼り付け               | `Ctrl` + `V`                 | `Cmd` + `V`                     | `Ctrl` + `V`                  |
| 切り取り               | `Ctrl` + `X`                 | `Cmd` + `X`                     | `Ctrl` + `X`                  |
| 元に戻す               | `Ctrl` + `Z`                 | `Cmd` + `Z`                     | `Ctrl` + `Z`                  |
| やり直し               | `Ctrl` + `Y` または `Ctrl` + `Shift` + `Z` | `Cmd` + `Shift` + `Z` | `Ctrl` + `Shift` + `Z`         |
| 全選択                 | `Ctrl` + `A`                 | `Cmd` + `A`                     | `Ctrl` + `A`                  |
| 検索                   | `Ctrl` + `F`                 | `Cmd` + `F`                     | `Ctrl` + `F`                  |
| 保存                   | `Ctrl` + `S`                 | `Cmd` + `S`                     | `Ctrl` + `S`                  |
| 名前を付けて保存       | `Ctrl` + `Shift` + `S`       | `Cmd` + `Shift` + `S`           | `Ctrl` + `Shift` + `S`        |
| 印刷                   | `Ctrl` + `P`                 | `Cmd` + `P`                     | `Ctrl` + `P`                  |
| 新しいウィンドウ       | `Ctrl` + `N`                 | `Cmd` + `N`                     | `Ctrl` + `N`                  |
| 新しいタブ             | `Ctrl` + `T`                 | `Cmd` + `T`                     | `Ctrl` + `T`                  |
| タブを閉じる           | `Ctrl` + `W`                 | `Cmd` + `W`                     | `Ctrl` + `W`                  |
| アプリケーションを終了 | `Alt` + `F4`                 | `Cmd` + `Q`                     | `Alt` + `F4`                  |

### ウィンドウ操作

| 操作                   | Windows                      | Mac                             | Linux                         |
|------------------------|------------------------------|---------------------------------|-------------------------------|
| ウィンドウを最大化     | `Win` + `↑`                  | `Cmd` + `Ctrl` + `F`            | `Super` + `↑`                 |
| ウィンドウを最小化     | `Win` + `↓`                  | `Cmd` + `M`                     | `Super` + `↓`                 |
| ウィンドウを左にスナップ | `Win` + `←`                 | `Cmd` + `Option` + `←`          | `Super` + `←`                 |
| ウィンドウを右にスナップ | `Win` + `→`                 | `Cmd` + `Option` + `→`          | `Super` + `→`                 |
| ウィンドウの切り替え   | `Alt` + `Tab`                | `Cmd` + `Tab`                   | `Alt` + `Tab`                 |
| 仮想デスクトップを開く | `Win` + `Tab`                | `Ctrl` + `↑`                    | `Super` + `S`                 |
| 次の仮想デスクトップ   | `Ctrl` + `Win` + `→`         | `Ctrl` + `→`                    | `Ctrl` + `Alt` + `→`          |
| 前の仮想デスクトップ   | `Ctrl` + `Win` + `←`         | `Ctrl` + `←`                    | `Ctrl` + `Alt` + `←`          |

### システムショートカット

| 操作                   | Windows                      | Mac                             | Linux                         |
|------------------------|------------------------------|---------------------------------|-------------------------------|
| 設定を開く             | `Win` + `I`                  | `Cmd` + `,`                     | `Super` + `,`                 |
| ファイルエクスプローラーを開く | `Win` + `E`                | `Cmd` + `Shift` + `C`           | `Super` + `E`                 |
| 検索を開く             | `Win` + `S` または `Win` + `Q` | `Cmd` + `Space`                | `Super` + `Space`             |
| タスクマネージャーを開く | `Ctrl` + `Shift` + `Esc`     | `Cmd` + `Option` + `Esc`       | `Ctrl` + `Shift` + `Esc`      |
| ロック画面             | `Win` + `L`                  | `Cmd` + `Ctrl` + `Q`            | `Super` + `L`                 |
| スクリーンショットを撮る | `Win` + `Shift` + `S`       | `Cmd` + `Shift` + `4`           | `Shift` + `PrtScn` または `Super` + `Shift` + `S` |

### ブラウザ操作

| 操作                   | Windows                      | Mac                             | Linux                         |
|------------------------|------------------------------|---------------------------------|-------------------------------|
| ページを再読み込み     | `Ctrl` + `R`                 | `Cmd` + `R`                     | `Ctrl` + `R`                  |
| キャッシュを無視して再読み込み | `Ctrl` + `Shift` + `R`       | `Cmd` + `Shift` + `R`           | `Ctrl` + `Shift` + `R`        |
| タブを再開             | `Ctrl` + `Shift` + `T`       | `Cmd` + `Shift` + `T`           | `Ctrl` + `Shift` + `T`        |
| 新しいタブを開く       | `Ctrl` + `T`                 | `Cmd` + `T`                     | `Ctrl` + `T`                  |
| タブを閉じる           | `Ctrl` + `W`                 | `Cmd` + `W`                     | `Ctrl` + `W`                  |
| 次のタブに移動         | `Ctrl` + `Tab`               | `Cmd` + `Option` + `→`          | `Ctrl` + `Tab`                |
| 前のタブに移動         | `Ctrl` + `Shift` + `Tab`     | `Cmd` + `Option` + `←`          | `Ctrl` + `Shift` + `Tab`      |
| アドレスバーにフォーカス | `Ctrl` + `L`                | `Cmd` + `L`                     | `Ctrl` + `L`                  |

### テキスト操作

| 操作                   | Windows                      | Mac                             | Linux                         |
|------------------------|------------------------------|---------------------------------|-------------------------------|
| 行の先頭に移動         | `Home`                       | `Cmd` + `←`                     | `Home`                        |
| 行の末尾に移動         | `End`                        | `Cmd` + `→`                     | `End`                         |
| 単語単位で左に移動     | `Ctrl` + `←`                 | `Option` + `←`                  | `Ctrl` + `←`                  |
| 単語単位で右に移動     | `Ctrl` + `→`                 | `Option` + `→`                  | `Ctrl` + `→`                  |
| 行を削除               | `Ctrl` + `Shift` + `K`       | `Cmd` + `Shift` + `K`           | `Ctrl` + `Shift` + `K`        |
| 行を複製               | `Ctrl` + `D`                 | `Cmd` + `D`                     | `Ctrl` + `D`                  |
| 行を上下に移動         | `Alt` + `↑` / `↓`            | `Option` + `↑` / `↓`            | `Alt` + `↑` / `↓`             |
| コメントアウト         | `Ctrl` + `/`                 | `Cmd` + `/`                     | `Ctrl` + `/`                  |

これらのショートカットキーは、Windows、Mac、Linuxそれぞれで一般的に使用されるものであり、効率的な操作をサポートします。各OS固有のキー配置を理解し、ショートカットを使いこなすことで、生産性を向上させることができます。
