## VSCode

以下に、Visual Studio Code (VSCode) の主要なショートカットキーを、Windows、Mac、Linuxそれぞれについて一覧にまとめました。

### ファイル操作

| 操作                   | Windows/Linux               | Mac                        |
|------------------------|-----------------------------|----------------------------|
| 新しいファイルを作成   | `Ctrl` + `N`                | `Cmd` + `N`                |
| 新しいウィンドウを開く | `Ctrl` + `Shift` + `N`      | `Cmd` + `Shift` + `N`      |
| ファイルを開く         | `Ctrl` + `O`                | `Cmd` + `O`                |
| ファイルを保存         | `Ctrl` + `S`                | `Cmd` + `S`                |
| 名前を付けて保存       | `Ctrl` + `Shift` + `S`      | `Cmd` + `Shift` + `S`      |
| ファイルを閉じる       | `Ctrl` + `W`                | `Cmd` + `W`                |
| フォルダーを開く       | `Ctrl` + `K`, `Ctrl` + `O`  | `Cmd` + `K`, `Cmd` + `O`   |
| 全てのファイルを閉じる | `Ctrl` + `K`, `Ctrl` + `W`  | `Cmd` + `K`, `Cmd` + `W`   |

### 編集

| 操作                   | Windows/Linux               | Mac                        |
|------------------------|-----------------------------|----------------------------|
| 切り取り               | `Ctrl` + `X`                | `Cmd` + `X`                |
| コピー                 | `Ctrl` + `C`                | `Cmd` + `C`                |
| 貼り付け               | `Ctrl` + `V`                | `Cmd` + `V`                |
| 元に戻す               | `Ctrl` + `Z`                | `Cmd` + `Z`                |
| やり直し               | `Ctrl` + `Y`                | `Cmd` + `Shift` + `Z`      |
| 行を削除               | `Ctrl` + `Shift` + `K`      | `Cmd` + `Shift` + `K`      |
| 行を複製               | `Alt` + `Shift` + `Down`    | `Option` + `Shift` + `Down`|
| 行を上下に移動         | `Alt` + `Up` / `Down`       | `Option` + `Up` / `Down`   |
| インデントを増やす     | `Ctrl` + `]`                | `Cmd` + `]`                |
| インデントを減らす     | `Ctrl` + `[`                | `Cmd` + `[`                |

### ナビゲーション

| 操作                      | Windows/Linux           | Mac                     |
|---------------------------|-------------------------|-------------------------|
| コマンドパレットを開く    | `Ctrl` + `Shift` + `P`  | `Cmd` + `Shift` + `P`   |
| クイックオープン          | `Ctrl` + `P`            | `Cmd` + `P`             |
| サイドバーの表示/非表示   | `Ctrl` + `B`            | `Cmd` + `B`             |
| 次のエディタへ移動        | `Ctrl` + `PageDown`     | `Cmd` + `Option` + `Right` |
| 前のエディタへ移動        | `Ctrl` + `PageUp`       | `Cmd` + `Option` + `Left`  |
| エディタを分割する        | `Ctrl` + `\`            | `Cmd` + `\`             |
| 次のターミナルへ移動      | `Ctrl` + `Shift` + `]`  | `Cmd` + `Shift` + `]`   |
| 前のターミナルへ移動      | `Ctrl` + `Shift` + `[`  | `Cmd` + `Shift` + `[`   |

### 検索と置換

| 操作                   | Windows/Linux             | Mac                      |
|------------------------|---------------------------|--------------------------|
| 検索                   | `Ctrl` + `F`              | `Cmd` + `F`              |
| 置換                   | `Ctrl` + `H`              | `Cmd` + `Option` + `F`   |
| 次を検索               | `F3`                      | `Cmd` + `G`              |
| 前を検索               | `Shift` + `F3`            | `Cmd` + `Shift` + `G`    |
| ファイル内を検索       | `Ctrl` + `Shift` + `F`    | `Cmd` + `Shift` + `F`    |
| 次の一致箇所に移動     | `Alt` + `Enter`           | `Cmd` + `Option` + `Enter` |

### ターミナル

| 操作                       | Windows/Linux               | Mac                        |
|----------------------------|-----------------------------|----------------------------|
| 新しいターミナルを開く     | `Ctrl` + `Shift` + `\``     | `Cmd` + `Shift` + `\``     |
| ターミナルの表示/非表示   | `Ctrl` + `\``               | `Cmd` + `\``               |
| ターミナルを分割する       | `Ctrl` + `Shift` + `5`      | `Cmd` + `Shift` + `5`      |
| 次のターミナルへ移動       | `Ctrl` + `Shift` + `]`      | `Cmd` + `Shift` + `]`      |
| 前のターミナルへ移動       | `Ctrl` + `Shift` + `[`      | `Cmd` + `Shift` + `[`      |
| ターミナルを閉じる         | `Ctrl` + `Shift` + `W`      | `Cmd` + `Shift` + `W`      |

### デバッグ

| 操作                    | Windows/Linux             | Mac                        |
|-------------------------|---------------------------|----------------------------|
| デバッグの開始/再開     | `F5`                      | `F5`                       |
| デバッグの一時停止      | `Shift` + `F5`            | `Shift` + `F5`             |
| 次の行をステップイン    | `F11`                     | `F11`                      |
| 次の行をステップアウト  | `Shift` + `F11`           | `Shift` + `F11`            |
| 次の行をステップオーバー| `F10`                     | `F10`                      |
| ブレークポイントの切替  | `F9`                      | `F9`                       |

### マルチカーソルと選択

| 操作                            | Windows/Linux                | Mac                         |
|---------------------------------|------------------------------|-----------------------------|
| 複数行選択                     | `Alt` + `Click`              | `Option` + `Click`          |
| 同じ単語を複数選択（次を追加） | `Ctrl` + `D`                 | `Cmd` + `D`                 |
| 同じ単語をすべて選択           | `Ctrl` + `Shift` + `L`       | `Cmd` + `Shift` + `L`       |
| 選択範囲を上下に移動           | `Alt` + `Up` / `Down`        | `Option` + `Up` / `Down`    |
| カーソルを行末に移動           | `End`                        | `Cmd` + `Right`             |
| カーソルを行頭に移動           | `Home`                       | `Cmd` + `Left`              |

### その他

| 操作                   | Windows/Linux               | Mac                        |
|------------------------|-----------------------------|----------------------------|
| 設定を開く             | `Ctrl` + `,`                | `Cmd` + `,`                |
| ファイルアイコンを表示/非表示 | `Ctrl` + `B`              | `Cmd` + `B`                |
| コメントアウト         | `Ctrl` + `/`                | `Cmd` + `/`                |
| ショートカットキー一覧 | `Ctrl` + `K`, `Ctrl` + `S`  | `Cmd` + `K`, `Cmd` + `S`   |

これらのショートカットキーを覚えることで、VSCodeの操作がより効率的になります。
